Linguagem SQL


A linguagem, SQL ajudará no projeto a acessar e manipular nossos registros no Banco de dados armazenando as informações do projeto de forma que possam ser usadas nas aplicações feitas.





Engenharia de Software.





 A disciplina Engenharia de Software que vai dar os caminhos necessários para o projeto relacionado a qualidade do software, desenvolvimento do programa garantindo maior desempenho e produtividade, e a pensarmos previamente nos custos de manutenção durante o desenvolvimento. 







Desenvolvimento de Sistemas. 





A importância da Disciplina de Desenvolvimento de Sistemas, será o conhecimento necessário para analisar, desenvolver, projetar e atualizar sistemas. Nesse projeto essa disciplina auxilia a criar sistemas informatizados e desenvolvimento para ampliar a capacidade do software.







Analise exploratória de dados. 





O Objetivo dessa disciplina no projeto é entender melhor os dados, identificar ideias sobre tendências e qualidade dos dados. A análise exploratória de dados nos ajudará a extrair informações relevantes de um conjunto de dados. 







Ambiente de Desenvolvimento e Operação. 





A importância da disciplina Devops no projeto será o trabalho em parceria, focado em um objetivo comum, além de automatizar as tarefas o Devops também acompanha e mede o desempenho das equipes para garantir a entrega no prazo e a qualidade dessa entrega. 

O conhecimento em Devops facilita a constante comunicação com várias áreas da Tecnologia, e assim nos facilitará  a aceitação a automatizar as tarefas e a adoção de novas Tecnologias ao projeto, implementando novas estratégias.
