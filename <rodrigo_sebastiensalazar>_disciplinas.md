Linguagem SQL:  
Utilizaremos a linguagem SQL como nossa principal ferramenta no que se diz respeito à arquitetura e manutenção do nosso banco de dados, também nos auxilia com a criação de relatórios para análise.  
  
Engenharia de Software:  
A engenharia de software nos guia com as diretrizes e o conhecimento necessário para criar um projeto de maneira organizada e eficiente, também colabora com a aprimoração das nossas habilidades de análise de problemas e levantamento de características de solução.  
  
Desenvolvimento de Sistemas:  
Esta disciplina nos ensina a desenvolver sites para a web, criando uma estrutura que possa servir de “ponte” entre o usuário final e as informações e funções do nosso banco de dados.  
  
Análise Exploratória de Dados:  
Nos proporciona técnicas e práticas diversificadas para extrair da melhor forma informações úteis para o nosso projeto, também proporciona métodos de organização para otimizar nossas análises.  
  
Ambiente de Desenvolvimento e Operação:  
Uma disciplina que implanta uma forma de trabalho mais colaborativa, onde devemos nos preocupar com os aspectos do projeto de forma organizada e que tenha um nível constate de comunicação entre os integrantes do projeto, também utilizando ferramentas em nuvem como o GitHub.
